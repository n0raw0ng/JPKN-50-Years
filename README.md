<h2 id="description">Description</h2>
<p>
  JPKN 50 Years is a dedicated web platform developed for an anniversary celebration ceremony. The primary goal of this project was to introduce a modern and innovative format for the official launching of the event, providing a fresh and contemporary experience for the audience.
</p>
<p>
  The site features a "System Locked" interface that requires users to enter a specific passcode for authentication. Upon successful entry, the system unlocks to reveal the official 50th-anniversary launching video.
</p>
